# Eat Recipe Importer

Local Chrome extension for importing web recipe links directly into Eat.

## Install locally

1. Open Chrome and go to `chrome://extensions`.
2. Turn on Developer mode.
3. Click Load unpacked.
4. Select this `chrome-extension` folder.

## Use

- Click the extension icon and connect with Google once.
- Keep the import target on Live for normal use.
- Switch the import target to Local when testing parser changes against `http://localhost:4174`.
- Open a recipe page.
- Click the Eat extension icon in Chrome's toolbar.
- Click Import current page.

Imported recipes are saved to Unfiled in Eat.
